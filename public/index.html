API Soundtracks

Documentation

Permet d'affichier, modifier ou de supprimer des bandes sonores à partir d'une base de donnée.
Les informations suivantes vous aiderons a naviguer cette API. Le lien du code Github.

API - Soundtracks

Routes

Obtenir toutes les bande sonores de la base de donnée.

GET /Soundtracks

http://www.votredomaine.com/Soundtracks

Paramètres possibles

orderBy

  • title
  • composer
  • label
  • year

direction

  • ascendant
  • descendant

limit

  • un chiffre entier

Accès à une bande sonore particulière.

GET /Soundtracks/{id}

http://www.votredomaine.com/Soundtracks

Paramètre requis: {id} un indentifiant d'une bande sonore

Ajout d'une nouvelle bande sonore.

POST /Soundtracks

http://www.votredomaine.com/Soundtracks

Paramètre requis: un objet sountracks dans le corps d'une rêquete.

const soundtracks = {
