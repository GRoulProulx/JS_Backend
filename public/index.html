API soundtracks

Documentation

Permet d'affichier, modifier ou de supprimer des bandes sonores à partir d'une base de donnée.
Les informations suivantes vous aiderons a naviguer cette API. Le lien du code Github.

API - soundtracks - Demo sur render

Routes

Récupération de toutes les bande sonores.

GET /soundtracks

http://www.votredomaine.com/soundtracks/init

Paramètres possibles:

http://www.votredomaine.com/soundtracks?=limit2&orderBy=composer&direction=asc

orderBy

  • title
  • composer
  • year

direction

  • ascendant
  • descendant

limit

  • un chiffre entier

Accès à une bande sonore particulière.

GET /soundtracks/{id}

Paramètre requis: {id} un indentifiant d'une bande sonore

http://www.votredomaine.com/soundtracks

Ajout d'une nouvelle bande sonore.

POST /soundtracks

http://www.votredomaine.com/soundtracks

Paramètre requis: un objet sountracks dans le corps d'une rêquete.

const soundtracks = {
title: "titre du film de la bande sonore",
composer: "compositeur",
year: année de sorti,
genre: ["genre 1", "genre 2", "genre 3"],
label: "maison de disques",
image: "image.jpg",
track: [
{
song: "Titre de la chanson",
link: "Lien youtube ou autre", }
}]

Paramètre requis:

  • un objet sountracks dans le corps d'une rêquete.

Suppression d'une bande sonore.

DELETE /soundtrack/{id}

Paramètre requis: l'indentifiant de la bande sonore.
